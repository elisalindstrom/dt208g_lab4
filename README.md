# DT208G Laboration 4
Applikation byggd med Angular som hämtar och visar kursdata från en webbtjänst.

## Länk
https://dt208g-lab4.onrender.com

## Installation
- Installera dependencies med kommando npm install.
- Öppna utvecklingsserver med kommando ng serve.
- Bygg med kommando ng build.

## Användning
- Kursdata hämtas från en extern JSON-fil och visas i en tabell.
- En service används för att hämta data från webbtjänst med HttpClient.
- Logik för filtrering och sortering av data hanteras med Signals i komponenten home.
- Det går att filtrera data utifrån kurskod och kursnamn samt sortera utifrån kurskod, kursnamn eller progression.
